------------------------------------------------------------------------------------------
METHOD DESCRIPTIONS
------------------------------------------------------------------------------------------

1. INDEPENDENT STUDENT (SI) - BASELINE
   - Standard supervised learning
   - No teacher, no distillation
   - Only cross-entropy loss with ground truth
   - Serves as baseline for measuring KD improvements

2. LOGIT MATCHING (LM) - PART 1
   - Student matches teacher's softened logits (soft targets)
   - Uses temperature-scaled softmax (T=4.0)
   - Loss = α*KL(soft_teacher || soft_student) + (1-α)*CE
   - Well-established, direct knowledge transfer

3. HINTS-BASED DISTILLATION (FitNet)
   - Transfers knowledge via intermediate feature maps
   - Aligns student and teacher hidden layer activations
   - Loss = α*MSE(adapted_student_feat || teacher_feat) + (1-α)*CE
   - Focuses on learning useful representations

4. CONTRASTIVE REPRESENTATION DISTILLATION (CRD)
   - Uses contrastive learning to align representations
   - Student learns to match teacher's representation space
   - Loss = α*Contrastive + (1-α)*CE
   - More sophisticated, learns semantic structure
